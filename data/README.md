# Data

## Скачивание
Скачайте `creditcard.csv` с Kaggle:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Поместите файл в `data/raw/creditcard.csv`

## Описание
- Time: секунды с первой транзакции
- V1-V28: PCA-трансформированные признаки
- Amount: сумма транзакции
- Class: 0 = нормальная, 1 = мошенническая (~0.17% аномалий)
